# 折叠面板功能使用指南

## 功能介绍

为了给正文内容更多的显示空间，搜索框和选择器现在可以折叠收起到标题栏中的一个按钮里。

## 为什么需要这个功能？

**问题**：搜索框和选择器占据了大量垂直空间，压缩了正文区域

**解决方案**：
- ✅ 可折叠的搜索和选择器面板
- ✅ 一键展开/收起
- ✅ 智能自动收起
- ✅ 更多空间用于阅读正文

## 使用方法

### 找到控制按钮

位置：右侧文本区域标题栏的右上角

**展开状态**：按钮显示 `▲ 收起`  
**收起状态**：按钮显示 `▼ 搜索/选择`

### 基本操作

**1. 收起面板**
```
点击 "▲ 收起" 按钮
→ 搜索框和选择器滑动隐藏
→ 正文区域扩大
```

**2. 展开面板**
```
点击 "▼ 搜索/选择" 按钮
→ 搜索框和选择器滑动显示
→ 自动聚焦搜索框
```

### 自动收起

为了提升用户体验，以下操作会自动收起面板：

**1. 通过搜索选择人物后**
```
输入搜索 → 点击结果 → 自动收起面板
→ 可以立即阅读传记
```

**2. 通过选择器选择人物后**
```
选择书卷 → 选择传记 → 选择人物 → 自动收起面板
→ 可以立即阅读传记
```


## 界面细节

### 按钮样式

**外观特点**：
- 半透明白色背景
- 白色边框
- 悬停时高亮
- 平滑过渡动画

**状态显示**：
- ▲ 向上箭头 = 当前展开（点击后收起）
- ▼ 向下箭头 = 当前收起（点击后展开）

### 动画效果

**展开动画**：
- 面板从0高度滑动到完整高度
- 淡入效果
- 持续时间：0.4秒

**收起动画**：
- 面板从完整高度滑动到0
- 淡出效果
- 持续时间：0.4秒

### 聚焦行为

展开面板时：
- ✅ 搜索框自动获得焦点
- ✅ 可以直接开始输入
- ✅ 无需额外点击

## 键盘快捷键（可选）

虽然当前版本使用鼠标点击，但可以添加键盘快捷键：

```javascript
// 在 TextView.vue 中添加
onMounted(() => {
  document.addEventListener('keydown', (e) => {
    // Ctrl + B 切换面板
    if (e.ctrlKey && e.key === 'b') {
      e.preventDefault()
      togglePanel()
    }
  })
})
```

## 常见问题

### Q: 面板收起后如何搜索？

A: 点击标题栏右上角的 "▼ 搜索/选择" 按钮，面板会展开并自动聚焦搜索框。

### Q: 能记住我的折叠状态吗？

A: 当前版本每次刷新页面都会重置为展开状态。
可以添加本地存储来记住状态：

```javascript
// 保存状态
watch(isPanelExpanded, (value) => {
  localStorage.setItem('panelExpanded', value.toString())
})

// 加载状态
onMounted(() => {
  const saved = localStorage.getItem('panelExpanded')
  if (saved !== null) {
    isPanelExpanded.value = saved === 'true'
  }
})
```

### Q: 为什么选择人物后会自动收起？

A: 这是为了优化用户体验：
- 选择完成后，你通常想立即阅读
- 自动收起给正文更多空间
- 减少手动操作步骤

如果不喜欢这个行为，可以注释掉代码中的：
```javascript
// 在 onPersonChange 中注释这一行
// isPanelExpanded.value = false
```

### Q: 面板收起时能使用选择器吗？

A: 不能，收起时选择器是隐藏的。
需要先点击展开按钮。

### Q: 能调整收起/展开的动画速度吗？

A: 可以，编辑 CSS：

```css
.collapsible-panel {
  transition: max-height 0.4s cubic-bezier(0.4, 0, 0.2, 1);
  /* 修改 0.4s 为你想要的时长 */
}
```
