# 人物路线图功能使用指南

## 功能介绍

根据传记文本中地点的时间顺序，自动在地图上绘制人物的行动轨迹，直观展示历史人物的活动路线。

## 核心功能

**自动解析**
- 从传记文本中提取所有地点
- 按照时间顺序排序
- 自动生成路线图

**可视化展示**
- 🔴 红色虚线连接各个地点
- 🔵 圆形编号标记显示顺序
- 🏷️ 地名标签清晰标注

**交互操作**
- 点击路线点查看详细信息
- 显示路线序号和总站数
- 自动调整地图视图

## 使用方法

### 基本操作

**1. 选择人物**
```
方式1：使用搜索框
输入人物名 → 选择结果

方式2：使用选择器
选择书卷 → 选择传记 → 选择人物
```

**2. 查看路线图**
```
选择人物后
→ 左侧地图自动绘制路线
→ 按时间顺序连接地点
→ 显示编号标记
```

**3. 查看路线点详情**
```
点击地图上的编号标记
→ 信息面板显示：
  - 地点名称
  - 路线位置（第X站/共Y站）
  - 时间
  - 阵营
  - 今名
```

## 路线排序规则

### 按年份排序

**提取年份：**
- 从data-year属性中获取
- 格式：155年、196年、200年等

**排序逻辑：**
```javascript
locations.sort((a, b) => {
  const yearA = parseInt(a.year) || 0
  const yearB = parseInt(b.year) || 0
  return yearA - yearB
})
```

**无年份处理：**
- 年份为空的地点排到最后
- 或设为0年处理

### 相同年份

如果多个地点年份相同：
- 保持在传记文本中的出现顺序
- 符合历史叙述逻辑

## 数据要求

### 传记文本格式

地名必须包含以下属性：

```html
<a class="place-name wei" 
   data-place="许昌" 
   data-year="196年" 
   data-faction="wei">许昌</a>
```

**必需属性：**
- `data-place` - 地点名称（必须在places.js中存在）
- `data-year` - 年份（用于排序）
- `data-faction` - 阵营（用于颜色）

### 地点数据

places.js中必须包含坐标：

```javascript
export const places = {
  '许昌': {
    lat: 34.035,
    lng: 113.852,
    modernName: '河南许昌市'
  }
}
```

## 常见问题

### Q: 为什么有些人物没有路线图？

A: 可能的原因：
1. 传记文本中没有地点标记
2. 只有一个地点（不绘制路线）
3. 地点没有年份信息（无法排序）

### Q: 路线图能否手动编辑？

A: 当前版本是自动生成的，不支持手动编辑。
路线完全基于传记文本中的地点和年份。

### Q: 如何添加更多路线点？

A: 在传记文本中添加地点标记：

```javascript
text: `
  <p>建安五年，太祖至
  <a class="place-name wei" 
     data-place="新地点" 
     data-year="200年" 
     data-faction="wei">新地点</a>
  。</p>
`
```

同时确保places.js中有该地点坐标。

### Q: 路线顺序错误怎么办？

A: 检查年份设置：
- 确保每个地点都有data-year
- 年份格式正确
- 年份数值符合历史顺序

### Q: 能否显示多个人物的路线对比？

A: 当前版本每次只显示一个人物的路线。
未来可以添加多人路线对比功能。

### Q: 点击传记中的地名后路线消失了？

A: 这是设计行为。点击传记地名会：
- 清除路线图
- 显示单个地点标记
- 聚焦到该地点

如果想再次查看路线，重新选择人物即可。

### Q: 路线连线能否改变颜色？

A: 可以，编辑MapView.vue中的drawRoute函数：

```javascript
routeLine = L.polyline(routeCoords, {
  color: '#your-color',  // 修改颜色
  weight: 3,             // 修改粗细
  dashArray: '10, 5'     // 修改虚线样式
})
```
