# 关于我们功能说明

## 功能介绍

在文本区域的底部添加了可展开的"关于我们"区域，用户可以点击查看项目介绍、功能说明、团队信息等内容。

## 位置和访问

**位置**：右侧文本区域的底部

**访问方式**：
1. 滚动到文本区域底部
2. 点击"关于我们"按钮
3. 区域展开显示详细内容

## 使用方法

### 展开内容

```
1. 滚动到文本区域底部
2. 看到"▼ 关于我们"按钮
3. 点击按钮
4. 内容向下展开（平滑动画）
```

### 收起内容

```
1. 内容展开状态下
2. 再次点击"▲ 关于我们"按钮
3. 内容向上收起（平滑动画）
```

### 阅读内容

```
展开后：
- 向下滚动阅读
- 点击相关链接
- 了解项目详情
```

## 自定义配置

### 修改按钮文字

```vue
<span class="about-title">关于我们</span>
<!-- 改为 -->
<span class="about-title">项目信息</span>
```

### 修改按钮位置

如果想放在顶部而非底部，调整HTML结构顺序。

### 修改展开动画

```css
.about-content {
  transition: max-height 0.4s cubic-bezier(0.4, 0, 0.2, 1);
  /* 修改 0.4s 为想要的时长 */
}
```

### 修改最大高度

```css
.about-content.expanded {
  max-height: 2000px;  /* 修改为足够大的值 */
}
```

### 修改颜色

```css
.about-toggle {
  background: linear-gradient(...);  /* 修改背景色 */
  color: var(--text-primary);        /* 修改文字色 */
}
```

## 常见问题

### Q: 内容太长，能否分页？

A: 当前版本使用滚动方式。如需分页：

```vue
<div v-if="currentPage === 1">第一页内容</div>
<div v-if="currentPage === 2">第二页内容</div>
```

### Q: 能否默认展开？

A: 可以，修改初始状态：

```javascript
const isAboutExpanded = ref(true)  // 改为 true
```

### Q: 能否添加图片？

A: 可以，在内容中添加：

```vue
<img src="/path/to/image.jpg" alt="描述" style="max-width: 100%; border-radius: 8px;" />
```

### Q: 能否添加视频？

A: 可以，但建议使用视频链接而非嵌入，避免影响性能。

### Q: 能否添加外部链接？

A: 可以：

```vue
<a href="https://example.com" target="_blank" style="color: var(--color-shu);">访问网站</a>
```

### Q: 如何统计展开次数？

A: 在toggleAbout函数中添加：

```javascript
const toggleAbout = () => {
  isAboutExpanded.value = !isAboutExpanded.value
  if (isAboutExpanded.value) {
    // 记录展开事件
    console.log('关于我们被展开')
  }
}
```
