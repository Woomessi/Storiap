# 搜索功能使用指南

## 功能介绍

现在你可以直接搜索人物名称，快速定位到相应的传记内容，无需手动选择书卷和传记。

## 使用方法

### 基本搜索

1. **找到搜索框**
   - 位于右侧文本区域的顶部
   - 标题下方，选择器上方
   - 带有🔍图标的输入框

2. **输入搜索内容**
   - 输入人物姓名，如"刘备"、"关羽"、"曹操"
   - 支持部分匹配，输入"刘"可以找到所有姓刘的人物
   - 不区分大小写

3. **查看搜索结果**
   - 输入后自动显示匹配结果
   - 显示人物名称、所属书卷和传记
   - 显示找到的结果数量

4. **选择结果**
   - 点击任意搜索结果
   - 自动跳转到该人物的传记
   - 搜索框自动关闭

5. **清除搜索**
   - 点击搜索框右侧的 ✕ 按钮
   - 或者删除所有输入内容
   - 或者点击搜索结果外的区域

## 搜索技巧

### 快速定位

**场景：** 想快速查看关羽的传记

**方法：**
1. 点击搜索框
2. 输入"关羽"
3. 点击搜索结果
4. 立即显示关羽传记

**优势：** 比手动选择"蜀书 → 关张马黄赵传第六 → 关羽"快得多！

### 探索人物

**场景：** 不确定有哪些姓张的人物

**方法：**
1. 输入"张"
2. 查看所有匹配结果
3. 选择感兴趣的人物

### 快速切换

**场景：** 连续查看多个人物

**方法：**
1. 查看完一个人物后
2. 再次点击搜索框
3. 输入下一个人物名
4. 无需重新选择书卷

## 界面细节

### 搜索框状态

| 状态 | 外观 | 说明 |
|------|------|------|
| 空闲 | 灰色边框 | 等待输入 |
| 聚焦 | 棕色边框，阴影 | 正在输入 |
| 有内容 | 显示清除按钮 | 可以清除 |

### 搜索结果状态

| 情况 | 显示内容 | 说明 |
|------|---------|------|
| 未输入 | 不显示 | 等待搜索 |
| 有结果 | 结果列表 | 点击选择 |
| 无结果 | 提示信息 | 换个关键词 |

### 视觉反馈

- **悬停效果**：鼠标悬停在结果上时，背景变浅色
- **选中动画**：点击后平滑过渡到传记内容
- **下拉动画**：搜索结果以滑入动画显示

## 常见问题

### Q: 为什么搜索不到某个人物？

A: 可能的原因：
1. 该人物尚未添加到数据库
2. 人物名称拼写错误
3. 使用了别名或字号（目前仅支持正名）

### Q: 如何搜索字或号？

A: 当前版本仅支持搜索人物正名。
未来可以扩展支持：
- 字（如"云长"搜索关羽）
- 号（如"孟德"搜索曹操）
- 别名

### Q: 能搜索传记标题吗？

A: 当前版本仅支持搜索人物名称。
如需按传记查找，请使用选择器。

### Q: 搜索结果如何排序？

A: 当前按照以下顺序：
1. 书卷顺序（魏→蜀→吴）
2. 传记在书中的顺序
3. 人物在传记中的顺序

### Q: 能记住搜索历史吗？

A: 当前版本不保存搜索历史。
这是一个很好的功能建议，可以在未来版本中添加。

### 添加字号搜索

```javascript
const aliasMap = {
  '云长': '关羽',
  '翼德': '张飞',
  '孟德': '曹操',
  // ... 更多别名
}

// 先查找别名

// 然后搜索真名
```

### 添加搜索历史

```javascript
import { ref, onMounted } from 'vue'

const searchHistory = ref([])

// 加载历史
onMounted(() => {
  const saved = localStorage.getItem('searchHistory')
  if (saved) {
    searchHistory.value = JSON.parse(saved)
  }
})

// 保存历史
const saveHistory = (query) => {
  if (!searchHistory.value.includes(query)) {
    searchHistory.value.unshift(query)
    searchHistory.value = searchHistory.value.slice(0, 10) // 保留10条
    localStorage.setItem('searchHistory', JSON.stringify(searchHistory.value))
  }
}
```

### 添加快捷键支持

```javascript
const handleKeyDown = (e) => {
  // Ctrl + F 聚焦搜索框
  if (e.ctrlKey && e.key === 'f') {
    e.preventDefault()
    searchInput.value?.focus()
  }
  
  // Esc 关闭搜索结果
  if (e.key === 'Escape') {

  }
  
  // Enter 选择第一个结果
  if (e.key === 'Enter' && searchResults.value.length > 0) {
    selectSearchResult(searchResults.value[0])
  }
}
```

## 与其他功能的配合

- 搜索并选择人物后
