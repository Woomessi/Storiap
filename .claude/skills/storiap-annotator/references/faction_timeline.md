# 三国势力版图时间线（阵营判断速查）

用于判断某地在某年应填 `wei` / `shu` / `wu` / `neutral`。

---

| 年份 | 事件 | 影响 |
|------|------|------|
| 189年前 | 东汉末年，诸侯割据前 | 全部用 `neutral` |
| 200年 | 官渡之战 | 曹操击败袁绍，逐步控制北方 |
| 208年 | 赤壁之战 | 三分天下格局开始形成 |
| 210-220年 | 荆州争夺 | 荆州归属多次易手（见下） |
| 220年 | 曹丕称帝建魏 | 魏国正式建立 |
  221年 | 刘备称帝建蜀汉 | 蜀汉正式建立 |
| 222年 | 孙权称吴王（229年称帝） | 吴国正式建立 |
| 263年 | 魏灭蜀 | 蜀汉亡 |
| 280年 | 晋灭吴 | 三国结束 |

## 通用规则

1. **189年以前**：全部 `neutral`（东汉统一时期）
2. **190-207年**：诸侯混战期，大部分地区用 `neutral`，曹操核心地区（许昌周边）可用 `wei`
3. **不确定时**：优先选 `neutral`，不要猜测错误的阵营

---

## 阵营 class vs data-faction 区别

```
class="place-name {传主阵营}"   ← 传记主角的阵营（整篇固定）
data-faction="{地点控制方}"     ← 该事件发生时该地实际控制方（随事件变化）
```

例：关羽传（shu）中，关羽攻打樊城（当时魏控）：
```html
<a class="place-name shu" data-place="樊城" data-year="219年" data-faction="wei">樊城</a>
```
